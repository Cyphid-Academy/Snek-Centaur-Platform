# @cyphid/snek-engine

Shared game engine for the Team Snek Centaur Platform.

Contains the domain type vocabulary and authoritative turn resolution logic for Team Snek. All three runtimes — SpacetimeDB, Centaur Server, and web clients — import from this package. No feature code exists yet; this is a typed skeleton.
