# @cyphid/snek-centaur-server-lib

Bot framework and server library for the Team Snek Centaur Platform.

Teams use this library to build their Centaur Server implementation. Provides the `Drive`/`Preference` abstractions, `defineBot` entry point, game invitation handling, healthcheck contract, and typed Convex component clients. No feature code exists yet; this is a typed skeleton.
