# @cyphid/convex-snek-platform

Convex Component for platform-wide state in the Team Snek Centaur Platform.

Owns the `users`, `centaur_teams`, `rooms`, `games`, `replays`, `api_keys`, and `webhooks` tables. Mounted by `@cyphid/snek-convex-host`. No feature code exists yet; this is a typed skeleton.
